Multiple Coordinate Systems
============================

A graph with multiple coordinate systems, each defined by a pair of X/Y axes,
can be digitized in Engauge.

Steps
-----

1. Obtain an image file showing multiple coordinate systems.

2. Import using **File > Import (Advanced)**:

   - Select the coordinate system count.
   - Click **OK**.

3. Select a coordinate system using the dropdown in the Coordinate System
   Toolbar.

4. Digitize the selected coordinate system using the same steps as for a
   single coordinate system.

5. Repeat steps 3–4 for each additional coordinate system.
