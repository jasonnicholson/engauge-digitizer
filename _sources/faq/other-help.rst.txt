Other Help Options
==================

Status Bar
----------

The status bar at the bottom of the application displays contextual hints to
help define axes or the scale bar.

Tooltip Help
------------

Placing the cursor over a toolbar button for a few seconds shows a brief
tooltip description.

What's This Help
----------------

Clicking the **What's This** button (``?``) and then any tool or window shows a
comprehensive help description for that item.

Contact
-------

Questions, comments, and requests can be sent to the author at the email
address shown in the **Help > About Engauge** dialog.

Debugging / Error Report
------------------------

Pressing **Ctrl+Shift+E** while the **Help > About Engauge** dialog is open
triggers a diagnostic error report, which can be helpful for bug reports.
