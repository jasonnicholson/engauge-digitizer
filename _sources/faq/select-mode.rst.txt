Select Mode Options
===================

In Select Mode, one or more points can be selected using a variety of options:

.. list-table::
   :header-rows: 1
   :widths: 30 40 30

   * - Option
     - Behavior
     - Useful For
   * - Click on unselected point
     - Selects that point; all already-selected points become unselected.
     - Selecting a single point
   * - Shift-click on unselected point
     - Selects that point while keeping already-selected points selected.
     - Adding one point to the selection
   * - Shift-click on selected point
     - Unselects that point while keeping other selected points selected.
     - Removing one point from the selection
   * - Click and drag
     - Selects all points within the drawn rectangle; already-selected points
       become unselected.
     - Selecting one or more points
   * - Ctrl+click and drag
     - Temporarily hides all points except those in the selected curve before
       drawing the selection rectangle. Already-selected points become
       unselected.
     - Selecting points in a single curve, or picking one point where
       multiple overlap
