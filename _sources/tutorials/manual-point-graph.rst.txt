Manual Point Graph Tutorial
============================

The following steps manually digitize a linear Cartesian graph with points along
one curve. These steps are essentially the same as
:doc:`manually digitizing a line graph <manual-line-graph>`.

1. Import the ``pointplot.bmp`` sample from the Samples directory using
   **File > Import**.

   .. image:: ../_images/tutormanpointgraph1.png
      :alt: File Import menu

2. Click the **Axes Point** button before entering axes points. These define the
   coordinate system.

   .. image:: ../_images/tutormanpointgraph2.png
      :alt: Axes Point button

3. Click on one of the axes to add the first axes point, then enter its graph
   coordinates.

   .. image:: ../_images/tutormanpointgraph3.png
      :alt: First axis point dialog

4. Click on one of the axes to add the second axes point, then enter its graph
   coordinates.

   .. image:: ../_images/tutormanpointgraph4.png
      :alt: Second axis point dialog

5. Click on one of the axes to add the third axes point, then enter its graph
   coordinates.

   .. image:: ../_images/tutormanpointgraph5.png
      :alt: Third axis point dialog

6. Click the **Curve Points** button before entering curve points. These will
   contain the digitized graph data.

   .. image:: ../_images/tutormanpointgraph6.png
      :alt: Curve Points button

7. Click on the image to add a curve point. Repeat until all points in the
   original image are digitized. For large point sets, consider using
   :doc:`automatic point matching <auto-point-graph>` instead.

   .. image:: ../_images/tutormanpointgraph7.png
      :alt: Curve points placed manually

8. Export the curve points using **File > Export As**.

   .. image:: ../_images/tutormanpointgraph8.png
      :alt: File Export As menu
