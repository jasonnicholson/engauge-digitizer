Floating Axes
=============

A graph with floating axes (where one or both axes have an unknown coordinate)
can be digitized in Engauge.

Steps
-----

1. Obtain an image file with floating axes.

2. Import using **File > Import (Advanced)**:

   - Select **4 points** for the axes point count.
   - Click **OK**.

3. Place four axes points and specify one coordinate value for each.
