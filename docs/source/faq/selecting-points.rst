Selecting Points
================

Selecting One or More Points
-----------------------------

To select axis or curve points with the Select Tool, use either:

1. Left-click on a single axis or curve point, then release.
2. Left-click and drag to select all curve points within the drawn rectangle,
   then release.

Successfully selected points each display a dashed square around them.

Editing One or More Points
--------------------------

1. Select the desired point(s) using the Select Tool (see above).
2. Right-click on one of the selected points.

.. note::

   - Axis and graph points cannot be edited at the same time; edit each type
     separately.
   - Right-clicking on an unselected point has no effect.
   - Double-clicking on any point has no effect.
