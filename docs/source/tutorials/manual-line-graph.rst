Manual Line Graph Tutorial
==========================

The following steps manually digitize a linear Cartesian graph with one curve.
These steps are essentially the same as
:doc:`manually digitizing a point graph <manual-point-graph>`.

1. Import the ``testcase.png`` sample from the Samples directory using
   **File > Import**.

   .. image:: ../_images/tutormanlinegraph1.png
      :alt: File Import menu

2. Click the **Axes Point** button before entering axes points. These define the
   coordinate system.

   .. image:: ../_images/tutormanlinegraph2.png
      :alt: Axes Point button

3. Click on one of the axes to add the first axes point, then enter its graph
   coordinates.

   .. image:: ../_images/tutormanlinegraph3.png
      :alt: First axis point dialog

4. Click on one of the axes to add the second axes point, then enter its graph
   coordinates.

   .. image:: ../_images/tutormanlinegraph4.png
      :alt: Second axis point dialog

5. Click on one of the axes to add the third axes point, then enter its graph
   coordinates.

   .. image:: ../_images/tutormanlinegraph5.png
      :alt: Third axis point dialog

6. Click the **Curve Points** button before entering curve points. These will
   contain the digitized graph data.

   .. image:: ../_images/tutormanlinegraph6.png
      :alt: Curve Points button

7. Click on the curve to add a curve point. Repeat until the graph is covered
   with a sufficient number of curve points. If this is too slow, consider using
   :doc:`segment fill <auto-line-graph>` instead.

   .. image:: ../_images/tutormanlinegraph7.png
      :alt: Curve points placed manually

8. Export the curve points using **File > Export As**.

   .. image:: ../_images/tutormanlinegraph8.png
      :alt: File Export As menu
