Manual Map Tutorial
====================

The following steps manually digitize a map using a scale bar.

1. Import the ``map.png`` sample from the Samples directory using
   **File > Import**.

2. Click the **Scale Bar** button to enter the scale bar. This defines the
   coordinate system for the map. (Unlike graph digitizing, a map uses a scale
   bar rather than three axis points.)

3. Click on one end of the scale bar in the image.

4. Drag to the other end of the scale bar in the image.

5. Enter the length of the scale bar in graph coordinates in the dialog that
   appears.

6. Click the **Curve Points** button before entering curve points.

7. Click on the map to add curve points. Repeat until all desired features are
   digitized.
